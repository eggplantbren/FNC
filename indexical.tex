Hybrid of FNC and SSA. Condition on the memory state but also, in the prior
information, assert that I will get {\em some} memory state, with a uniform
prior over conscious beings.

The hypothesis space under consideration is
\begin{align}
\overbrace{\{0, 1, 2, ...\}}^{N_*} \times
\overbrace{[0, 1]}^{\theta} \times
\overbrace{[0, 1]}^{\phi} \times
\overbrace{\{0, 1, 2, ...\}^{N_*}}^{\{n_i\}_{i=1}^{N_*}}.
\end{align}

For now assume $N^*$ is known.
The prior over the rest of the hypothesis space is
\begin{align}
p(\theta, \phi, \boldsymbol{n}) &\propto
    \frac{1}{\theta\phi}\prod_{i=1}^{N_*} p(n_i \given \theta, \phi).
\end{align}
For arguments' sake we will set the lower bounds on $\theta$ and $\phi$
to $\exp(-50)$.

Now, the probability distribution for the number of beings at each
star is some sort of zero-inflated distribution $f(n_i)$, with
where $f(0) = 1 - \theta\phi$ and we assume $f(n)$ has fixed shape
for $n \geq 1$ that does not depend on $\theta$ and $\phi$.

The probability that my memories exist (in this simple model) will be
proportional to the number of consciousnesses,
\begin{align}
P(\textnormal{Brendon exists} \given \theta, \phi, \boldsymbol{n})
    &= \epsilon\sum_{i=1}^{N_*} n_i.
\end{align}
This extends the space with the binary variable of my existence, so the
full joint is now
\begin{align}
p(\theta, \phi, \boldsymbol{n}, \textnormal{Brendon exists}) &\propto
    \left(\frac{1}{\theta\phi}\prod_{i=1}^{N_*} p(n_i \given \theta, \phi)\right)
    \left(\epsilon\sum_{i=1}^{N_*} n_i\right).
\end{align}

If we marginalise over $\boldsymbol{n}$, we get
\begin{align}
p(\theta, \phi, \textnormal{Brendon exists}) &\propto
    \frac{1}{\theta\phi}
    \sum_{n_1, ..., n_{N_*}}
    \left(\prod_{i=1}^{N_*} p(n_i \given \theta, \phi)\right)
    \left(\epsilon\sum_{i=1}^{N_*} n_i\right).
\end{align}
This looks ugly, but simplifies to the conditional expectation of the total
$\sum_i n_i$ given the parameters:
\begin{align}
p(\theta, \phi, \textnormal{Brendon exists}) &\propto
    \frac{1}{\theta\phi}
    \theta\phi E \epsilon,
\end{align}
where $E$ is the expectation of $n_i$ given that $n_i \geq 1$.
This is the strong FNC conclusion of abundant life.

However, if I take the data as something different --- I am going to get
{\em some} memories, so the space of possible datasets is not
$\{\textnormal{Brendon exists}, \textnormal{Brendon does not exist}\}$
but is instead the set of possible memories, there is an additional factor.
Instead we get 
\begin{align}
p(\theta, \phi, \boldsymbol{n}, \textnormal{Brendon exists},
  \textnormal{Brendon is returned}) &\propto
    \left(\frac{1}{\theta\phi}\prod_{i=1}^{N_*} p(n_i \given \theta, \phi)\right)
    \left\{
        \begin{array}{lr}
         \frac{\epsilon\sum_{i=1}^{N_*} n_i}{\sum_{i=1}^{N_*} n_i}, & \sum_i n_i \geq 1 \\
        0, & \textnormal{otherwise}
        \end{array}
    \right.\\
    &= \left(\frac{1}{\theta\phi}\prod_{i=1}^{N_*} p(n_i \given \theta, \phi)\right)
    \epsilon\mathds{1}\left(\sum_{i=1}^{N_*} n_i \geq 1\right).
\end{align}
Marginalising over $\boldsymbol{n}$ now gives
\begin{align}
p(\theta, \phi, \textnormal{Brendon exists},
  \textnormal{Brendon is returned}) &\propto
    \frac{1}{\theta\phi}
    \left(1 - (1 - \theta\phi)^{N_*}\right)
\epsilon.
\end{align}
I will call this the full indexical conditioning (FIC) likelihood.
